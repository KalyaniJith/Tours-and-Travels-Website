# Tours and Travels Website

## Overview

This is a simple Tours and Travels website built using HTML. The website showcases domestic and international tour packages along with travel destination information and ticket booking links.

## Features

* Domestic Tour Packages

  * Kashmir Tour Package
* International Tour Packages

  * Paris Tour Package
* Flight Ticket Booking Links
* Bus Ticket Booking Links
* Train Ticket Booking Links
* Travel destination images

## Technologies Used

* HTML5
